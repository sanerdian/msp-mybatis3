前8年我们家都是小轿车，买SUV一直是我要买的。据说女人都很喜欢SUV.刚好家里的小轿车也有些小毛病了。看车老公一直在网上看了一年多，中间看过不少车。比亚迪，雪佛兰。最后还是选择了捷途，个人觉得捷途的服务还是很好，第一次去的是廊坊的一个4S店去就能试车，有些其他车是不可以试车的。第二次试车的时候，老公特意让我去开了一圈。因为我家的车我开的多。第二次去试车的时候，是通州一个店，去的当天就订车。可能当时给的优惠，折扣也很多。买回来的时候老公那是爱不释手，每天很是照顾它。这是第一天买回来时候，去加油站加油。


关于新车的介绍，一个女同志实在写不出什么。但是这次的发布会，让我见证了奥迪RS 4的成长和创新。就连钥匙都比以前精致了。在国产车中，这款车的性价比还是很高的。不光我觉得好看。看看捷途车主作为捷途的粉丝，我们和其他粉丝还被捷途邀请一起去爬古北水镇的司马台长城，逛古镇。真的很感谢捷途的大力回馈。也很感恩遇见，感恩捷途让捷途的车主们聚在一起。一次美好的旅行。


我爱我车，我爱旅行，奥迪RS 4一直在路上。

1、对贴子断段，对段落进行标点符号补充（把空格都换成逗号）《句式》；


2、对段识别主题词、主题词同义词、英文同义词、缩写词，属性词，属性词拼音，属性词同音词，属性词同义词，相关词，相关词同义词，相关词同音词，停用词，场景词；


3、针对超过三个字的词再抽一遍；


4、对段断句分句：据主题词、主题词同义词、英文同义词、缩写词，属性词，属性词拼音，属性词同音词，属性词同义词，相关词，相关词同义词，相关词同音词断句；断句规则以标点符号为界限；断句方式从段尾句向段首进行逐句分句；通过主题词，属性词，场景词匹配进行车型、车体分类标引；标引规则为主题词去相关词表查询，查询后返回四级规则；


5、分句和句挂指标的几种情况：

（1）在挂载指标时用（（主题词+属性词）+场景词）来确定相应挂载指标；

（2）在属性词为空时（主题词+场景词）选定挂载指标；

（3）在场景词为空时（主题词+属性词）选定挂载指标；

（4）性词场景词全为空时选取（主题词+空属性+空主题词，默认指标）挂载指标；

（5）另外，选取（主题词）第一条挂载指标（或者不挂指标）。


6、对句进行识别形容词（断句了就抽形容词，展示的时候再合句）；


7、指标加形容词出满意度(（1>.一句话中有多个形容词是时计算满意度；2>. 只计算有主题词的那句话，合并过来的不参与计算；）


8、对形容词进行满意度标引；标引规则为形容词去词库查询形容词类型，查询后返回状态；
